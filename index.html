Features

핵심 기능

사용자 경험을 최우선으로 설계된 기능들로 모임 준비의 모든 과정을 혁신합니다.

스마트 일정 조율

  • 투표 링크 공유로 간편 참여
  • 가능 일정 자동 집계
  • 최적 날짜 AI 추천
  • 캘린더 연동 (예정)

중간지점 추천

  • 각자 출발지 입력
  • 대중교통 시간 기반 계산
  • 공정한 중간 위치 제안
  • 이동 시간 미리보기

맞춤 장소 추천

  • 모임 목적별 필터 (회식, 데이트 등)
  • 인원수 고려 추천
  • 평점/리뷰 기반 정렬
  • 예약 연동 (예정)

자동 정산 (예정)

  • 영수증 촬영으로 자동 입력

  • 송금 요청 알림
  • 정산 내역 기록

Business Model

수익 모델

초기에는 사용자 확보에 집중하고, 트래픽 기반의 광고 수익 모델을 중심으로 수익화를 진행합니다.

수익화 전략
  • 1단계: 사용자 확보 (무료)
  • 2단계: 광고 수익 (배너/네이티브)
  • 3단계: 제휴 수수료 (장소 예약)
