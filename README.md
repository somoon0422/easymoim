# 이지모임 (EasyMoim)

> AI 기반 스마트 약속 조율 서비스 - 모임의 A부터 Z까지, 당신의 모임비서

이지 모임 : [prototypeeasymoim.vercel.app](https://prototypeeasymoim.vercel.app/)

## 서비스 소개

**100개 넘는 메시지 정리하지 말고, AI가 3분 만에 자동으로 끝내드립니다.**

이지모임은 그룹 채팅에서의 복잡한 일정 조율 과정을 자동화하는 서비스입니다. 바쁜 현대인들이 모임 준비에 쓰는 시간을 획기적으로 줄여, 모임 자체에 집중할 수 있도록 도와드립니다.

---

## 핵심 기능

| 기능 | 설명 |
|------|------|
| **간편한 일정 조율** | 투표 기능으로 모든 참여자의 일정을 한눈에 파악 |
| **AI 장소 추천** | 조건에 맞는 최적 장소 자동 추천 |
| **실시간 소통** | 모임 채팅으로 빠른 의사결정 지원 |

### 1. 간편한 일정 조율
- 투표 기능으로 모든 참여자의 일정을 한눈에 파악
- AI가 최적의 날짜/시간 자동 추천
- 링크 공유만으로 간편하게 참여

### 2. AI 장소 추천
- 참여자들의 출발지 기반 중간지점 계산
- 모임 목적에 맞는 최적 장소 자동 추천
- 맛집, 카페, 회의실 등 다양한 카테고리 지원

### 3. 실시간 소통
- 모임 채팅으로 빠른 의사결정 지원
- 알림 기능으로 중요 정보 놓치지 않기

### 4. 자동 정산 (예정)
- 1/N 자동 계산
- 송금 요청 알림
- 정산 내역 기록 및 관리
